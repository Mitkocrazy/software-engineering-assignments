# Задание 3: Реализация, Git и качество на кода

## 1. Реализирана функционалност

- Регистрация на потребители
- Вход в системата
- CRUD операции за публикации
- Основна навигация между страниците


---

## 2. Как работи

1. Потребителят се регистрира или влиза
2. Получава достъп до системата
3. Може да създава публикации
4. Данните се записват в базата
5. Публикациите се визуализират в списък


---

## 3. Възникнали проблеми

1. Проблем с автентикацията (JWT токен не се запазваше правилно)
2. Грешки при връзката между frontend и backend
3. Неправилно обновяване на данни след CRUD операции


---

## 4. Как са решени

1. Добавено правилно съхранение на токена (localStorage)
2. Проверка на API endpoint-и и корекция на URL адресите
3. Добавено презареждане на данни след операции


---

## 5. Следващи стъпки

- Подобряване на UI
- Добавяне на коментари
- Добавяне на роли и права
- Тестване и оптимизация
