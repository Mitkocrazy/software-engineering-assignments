## 3. Описани дефекти

### Bug 1
**Заглавие:** Невалиден вход при празни данни  
**Стъпки:**
1. Отиди на login
2. Остави празни полета
3. Натисни login  

**Очакван резултат:** Грешка  
**Реален резултат:** Системата допуска вход  
**Статус:** Поправен  

---

### Bug 2
**Заглавие:** Пост не се обновява веднага  
**Стъпки:**
1. Редактирай пост
2. Запази  

**Очакван резултат:** Обновен пост  
**Реален резултат:** Старият остава до refresh  
**Статус:** Поправен
